Unit 1: Introduction to Machine Learning (ML)

In this unit, I explore the basic concepts of machine learning, including supervised and unsupervised learning, algorithms, and their applications in real-world scenarios.

Key Takeaways: Overview of ML models, key algorithms like decision trees, and ethical considerations in ML applications.

Unit 2: Exploratory Data Analysis

This unit focused on exploratory data analysis techniques, including data visualization, statistical summaries, and identifying trends or patterns in datasets.

Key Tools: Matplotlib, Pandas for data analysis, and Seaborn for visualization.

Unit 3: Correlation and Regression

Here, I learned about correlation analysis, simple and multiple regression models, and their applications in predicting relationships between variables.

Applications: Predicting relationships between factors such as age and income.

Unit 4: Linear Regression with Scikit-Learn

This unit introduced the Scikit-Learn library for implementing linear regression models and evaluating their performance through metrics like mean squared error and R-squared values.

Unit 5: Clustering

An introduction to clustering techniques, such as K-Means, and their use in grouping similar data points without labeled outcomes.

Key Concept: Using K-Means clustering for customer segmentation based on purchase behaviors.

Unit 6: Clustering with Python

In this unit, I implemented clustering techniques using Python, focusing on K-Means, hierarchical clustering, and DBSCAN.

Unit 7: Introduction to Artificial Neural Networks (ANNs)

This unit provided an introduction to artificial neural networks (ANNs) and their architectures, including feedforward networks and backpropagation for training.

Unit 8: Training an Artificial Neural Network

Hands-on implementation of training a neural network using popular libraries like TensorFlow and Keras, focusing on adjusting weights and bias to improve accuracy.

Unit 9: Introduction to Convolutional Neural Networks (CNNs)

This unit covered the fundamentals of convolutional neural networks, specifically their use in image classification tasks.

Unit 10: CNN Interactive Learning

Engaging with interactive exercises to deepen my understanding of CNNs and their applications in pattern recognition and image processing.

This unit focused on choosing the appropriate machine learning model and evaluating it using performance metrics like accuracy, precision, and recall.
